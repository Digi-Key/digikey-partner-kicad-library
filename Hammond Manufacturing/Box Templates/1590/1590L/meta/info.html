Project template containing the recommended board layout for Hammond Manufacturing 1590L:
Unpainted: 1590LF - HM3616-ND
Black: 1590LBK - HM3615-ND
Unpainted: 1590L - HM1924-ND


Features:

The "Dwgs.User" layer contains all critical dimmensions in milimeters[inches].
The "Edge.Cuts" layer contains the maximum board outline and holes to align with board standoffs.
