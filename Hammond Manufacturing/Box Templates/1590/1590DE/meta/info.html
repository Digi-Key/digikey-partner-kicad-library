Project template containing the recommended board layout for Hammond Manufacturing 1590DE:
Black: 1590WDEBK - HM483-ND
Unpainted: 1590WDE - HM482-ND
Black: 1590DEBK - HM440-ND
Unpainted: 1590DE - HM439-ND


Features:

The "Dwgs.User" layer contains all critical dimmensions in milimeters[inches].
The "Edge.Cuts" layer contains the maximum board outline and holes to align with board standoffs.
