Project template containing the recommended board layout for Hammond Manufacturing 1590BBS:
Natural: 1590WBBS - HM2295-ND
Black: 1590BBSBK - HM3573-ND
Unpainted: 1590BBS - HM1902-ND


Features:

The "Dwgs.User" layer contains all critical dimmensions in milimeters[inches].
The "Edge.Cuts" layer contains the maximum board outline and holes to align with board standoffs.
