Project template containing the recommended board layout for Hammond Manufacturing 1590TRPC:
Yellow: 1590TRPCYL - HM1249-ND
Red: 1590TRPCRD - HM471-ND
Purple: 1590TRPCPR - HM494-ND
Orange: 1590TRPCOR - HM470-ND
Gray: 1590TRPCLG - HM469-ND
Green: 1590TRPCGR - HM468-ND
Blue: 1590TRPCCB - HM467-ND
Black: 1590TRPCBK - HM466-ND


Features:

The "Dwgs.User" layer contains all critical dimmensions in milimeters[inches].
The "Edge.Cuts" layer contains the maximum board outline and holes to align with board standoffs.
