Project template containing the recommended board layout for Hammond Manufacturing 1590STPC:
Yellow: 1590STPCYL - HM491-ND
Red: 1590STPCRD - HM457-ND
Purple: 1590STPCPR - HM498-ND
Orange: 1590STPCOR - HM456-ND
Gray: 1590STPCLG - HM455-ND
Green: 1590STPCGR - HM454-ND
Blue: 1590STPCCB - HM453-ND
Black: 1590STPCBK - HM452-ND
Unpainted: 1590STPC - HM451-ND


Features:

The "Dwgs.User" layer contains all critical dimmensions in milimeters[inches].
The "Edge.Cuts" layer contains the maximum board outline and holes to align with board standoffs.
