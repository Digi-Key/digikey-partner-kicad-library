Project template containing the recommended board layout for Hammond Manufacturing 1557H:
Gray: 1557HGY - 164-1557HGY-ND
Black: 1557HBK - 164-1557HBK-ND
Gray: 1557HAGY - 164-1557HAGY-ND
Black: 1557HABK - 164-1557HABK-ND


Features:

The "Dwgs.User" layer contains all critical dimmensions in milimeters[inches].
The "Edge.Cuts" layer contains the maximum board outline and holes to align with board standoffs.
